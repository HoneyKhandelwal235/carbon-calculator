# Deployment Guide

See the comprehensive deployment guide in the artifacts folder for detailed instructions on deploying your Carbon Calculator to the web.

## Quick Start Options

### 1. GitHub Pages (Recommended)
1. Create a GitHub account at [github.com](https://github.com)
2. Create a new repository named `carbon-calculator`
3. Upload all project files
4. Enable GitHub Pages in Settings â†’ Pages
5. Your site will be live at `https://YOUR-USERNAME.github.io/carbon-calculator/`

### 2. Netlify (Easiest)
1. Go to [netlify.com](https://www.netlify.com)
2. Drag and drop your project folder
3. Site is live instantly!

### 3. Vercel (Fastest)
1. Go to [vercel.com](https://vercel.com)
2. Import from GitHub
3. Auto-deploy on every update

## Important Files

- `sitemap.xml` - Helps search engines index your site
- `robots.txt` - Guides search engine crawlers
- `README.md` - Project documentation

## After Deployment

1. Submit your site to [Google Search Console](https://search.google.com/search-console)
2. Update the sitemap.xml with your actual URL
3. Update robots.txt with your actual URL
4. Share your site on social media!

For detailed step-by-step instructions, see the full deployment guide in the artifacts.
